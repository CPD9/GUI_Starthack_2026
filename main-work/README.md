# Main Work

Main app code lives here.

## Folders

- `web` - Next.js 15 app (React, Better Auth, Polar)
- `mcp-server` - local MCP server (TypeScript)
- `app` - legacy Expo starter kept for reference

## Start

Web app:

- `cd web`
- `cp .env.example .env.local`
- `npm run dev`

MCP server:

- `cd mcp-server`
- `npm run dev`
