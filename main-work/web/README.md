# Web App

Next.js 15 app for the StartHack 2026 analytics assistant.

## Run

1. Copy env template:
   - `cp .env.example .env.local`
2. Install dependencies:
   - `npm install`
3. Start dev server:
   - `npm run dev`

## Scripts

- `npm run dev`
- `npm run build`
- `npm run start`
- `npm run lint`
- `npm run typecheck`
- `npm run db:generate`
- `npm run db:push`
- `npm run inngest:dev`
